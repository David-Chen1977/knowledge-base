# 电力新能源 · 关键术语表

**用途：** PE/VC 一级市场投资研究参考

---

## 一、光伏

| 术语 | 中文 | 说明 |
|------|------|------|
| PV | 光伏 | Photovoltaic，光生伏特效应发电 |
| PERC | 钝化发射极背面电池 | Passivated Emitter and Rear Cell，当前主流 P 型电池技术 |
| TOPCon | 隧穿氧化层钝化接触电池 | Tunnel Oxide Passivated Contact，当前主流 N 型电池，量产效率 25-26% |
| HJT / HIT | 异质结电池 | Heterojunction Technology，效率 25.5-26.5%，工艺步骤少但设备贵 |
| BC / IBC | 背接触电池 | Back Contact，正面无栅线，美观+高效率，隆基主力方向 |
| Perovskite | 钙钛矿太阳能电池 | 下一代薄膜电池技术，实验室效率>33%（叠层），稳定性攻关中 |
| BIPV | 建筑光伏一体化 | Building Integrated PV，光伏组件作为建筑材料 |
| BAPV | 建筑附着光伏 | Building Attached PV，光伏组件附着于既有建筑 |
| LCOE | 平准化度电成本 | Levelized Cost of Electricity，光伏发电全生命周期度电成本 |
| 硅料/多晶硅 | Polysilicon | 光伏产业链最上游原料，西门子法为主，颗粒硅新技术 |
| 金刚线 | Diamond Wire | 硅片切割用线，线径更细→出片更多→成本更低 |
| N 型 vs P 型 | N/P 型硅片 | N 型（磷掺杂）效率更高、衰减更低，当前替代 P 型中 |
| 双面率 | Bifaciality | 双面发电组件背面/正面功率比 |
| 衰减率 | Degradation | 光伏组件每年发电效率下降比例（P型首年2-3%，N型<1%） |

## 二、储能

| 术语 | 中文 | 说明 |
|------|------|------|
| LFP | 磷酸铁锂电池 | LiFePO4，当前储能主力电化学路线 |
| NCM/NCA | 三元锂电池 | 能量密度高，用于动力电池，储能较少 |
| 钠离子电池 | Sodium-ion Battery | 钠替代锂，原料丰富成本低，能量密度偏低，2025-2026量产元年 |
| 全钒液流电池 | Vanadium Redox Flow Battery (VRFB) | 长时储能（4-12h），循环寿命>20年，但钒成本高 |
| 铁铬液流电池 | Iron-Chromium Flow Battery | 原料更便宜，能量密度更低，国家电投主导 |
| 压缩空气储能 | CAES | Compressed Air Energy Storage，大规模长时储能，选址依赖洞穴 |
| 飞轮储能 | Flywheel Energy Storage | 短时高频响应，用于调频 |
| 超级电容 | Supercapacitor | 短时大功率，秒级响应 |
| BMS | 电池管理系统 | Battery Management System，监控电压/温度/SOC/SOH |
| EMS | 能量管理系统 | Energy Management System，充放电调度策略 |
| PCS | 储能变流器 | Power Conversion System，AC/DC 双向变换 |
| SOC | 荷电状态 | State of Charge，电池剩余电量百分比 |
| SOH | 健康状态 | State of Health，电池老化程度 |
| DOD | 放电深度 | Depth of Discharge，充放电区间 |
| C-rate | 充放电倍率 | 1C = 1小时充满/放完 |
| RTE | 往返效率 | Round-Trip Efficiency，充放电能量转换效率 |
| 独立储能 | Standalone Storage | 独立参与电力市场的储能电站 |
| 容量电价 | Capacity Payment | 为可用容量付费，保障储能收益的机制 |

## 三、风电

| 术语 | 中文 | 说明 |
|------|------|------|
| LCOW | 平准化度电成本（风电） | Levelized Cost of Wind |
| 陆上风电 | Onshore Wind | 陆地上安装，当前LCOE已低于火电 |
| 海上风电 | Offshore Wind | 海洋中安装，利用更高风速，LCOE仍在下降通道 |
| 浮式风电 | Floating Offshore Wind | 深水区（>60m）使用浮式基础，早期技术 |
| 单桩 | Monopile | 固定式基础，用于浅海 |
| 导管架 | Jacket | 固定式基础，用于较深海 |
| DFIG | 双馈异步发电机 | Doubly-Fed Induction Generator，当前主流风机技术 |
| 直驱永磁 | Direct Drive PMSG | 无齿轮箱，可靠性高，金风/海装等主力 |
| 容量因子 | Capacity Factor | 实际发电/理论最大发电，海上风电 40-55% |
| 叶片回收 | Blade Recycling | 退役叶片（玻纤/碳纤复合材料）回收利用 |
| 风功率预测 | Wind Power Forecasting | AI/ML 预测风功率，电网调度需求 |

## 四、氢能

| 术语 | 中文 | 说明 |
|------|------|------|
| 灰氢 | Grey Hydrogen | 天然气重整制氢，有碳排放 |
| 蓝氢 | Blue Hydrogen | 灰氢+CCUS（碳捕集），低碳 |
| 绿氢 | Green Hydrogen | 可再生能源电解水制氢，零碳 |
| PEM 电解槽 | Proton Exchange Membrane | 质子交换膜电解水，响应快，效率高，贵金属催化剂 |
| AEM 电解槽 | Anion Exchange Membrane | 阴离子交换膜电解水，非贵金属，研发阶段 |
| SOEC | 固体氧化物电解槽 | Solid Oxide Electrolyzer，高温电解，效率最高 |
| LOHC | 液态有机氢载体 | Liquid Organic Hydrogen Carrier，常温常压储运氢 |
| 氢脆 | Hydrogen Embrittlement | 金属在氢气中韧性下降，材料科学难题 |
| FCV / FCEV | 燃料电池汽车 | Fuel Cell Electric Vehicle |
| 膜电极 | MEA | Membrane Electrode Assembly，燃料电池核心部件 |
| 双极板 | Bipolar Plate | 燃料电池电堆组成部分，金属/石墨两种路线 |
| CCUS | 碳捕集利用与封存 | Carbon Capture, Utilization and Storage |

## 五、智能电网/虚拟电厂/电力市场

| 术语 | 中文 | 说明 |
|------|------|------|
| VPP | 虚拟电厂 | Virtual Power Plant，聚合分布式资源参与电力市场 |
| DER | 分布式能源资源 | Distributed Energy Resource，包括分布式光伏/储能/充电桩 |
| 需求侧响应 | Demand Response (DR) | 用户侧根据电价信号调整用电行为 |
| 辅助服务 | Ancillary Services | 调频/调峰/备用/黑启动等保障电网稳定服务 |
| 现货市场 | Spot Market | 日前+实时，即期电力交易市场 |
| 中长期市场 | Forward Market | 月度/季度/年度电力合约交易 |
| 绿证 | Green Electricity Certificate (GEC) | 可再生能源发电环境属性凭证 |
| 微电网 | Microgrid | 局部发配用一体化，可离网运行 |
| AMI | 高级量测体系 | Advanced Metering Infrastructure，智能电表+通信网络 |
| 配电自动化 | Distribution Automation | 配网侧远程监控/故障定位隔离恢复 |
| 电力物联网 | Power IoT | 电网侧广泛部署的传感/通信/边缘计算 |
