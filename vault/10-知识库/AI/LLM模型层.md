## 二、商业模式演进

| 阶段 | 模式 | 典型定价 | 利润率 | 代表 |
|------|------|---------|-------|------|
| 1.0 API 按量 | 按 Token 计费 | ¥0.1-1/百万 Token | 极低（价格战 >90% 降幅） | 智谱/百度/DeepSeek |
| 2.0 企业私有化 | 买断+年度服务费 | ¥100 万-2000 万/单 | 高（但交付重） | 智谱/百川 |
| 3.0 C 端订阅 | 月费 ¥19-99 | ¥30/月均 | 中 | Kimi/豆包 |
| 4.0 Agent 分成 | 按效果/交易额比例 | 未知 | 潜力高 | 月之暗面 Kimi Claw |

**趋势：** API 已商品化（毛利率走低），企业私有化是当前收入主力但不可扩展，Agent 是模型层商业化的"第二曲线"。

## 三、技术趋势（2026-2028）

### 3.1 多模态融合
纯文本大模型已成为历史。文本+图像+语音+视频+3D 的全模态能力是 2026 年标配。MiniMax 以语音/视频领先，阶跃星辰押注视频生成。

### 3.2 超长上下文
| 模型 | 窗口 | 差异 |
|------|------|------|
| Kimi | 200 万字 | 中国首创，C 端产品力强 |
| DeepSeek V4 | 100 万 Token | 标配 |
| Gemini 1.5 Pro | 100 万 Token | Google 生态 |

### 3.3 推理能力升级
"快思考"→"慢思考"：DeepSeek R1/R2 的 Chain of Thought + 强化学习推理代表了新一代架构方向。OpenAI o1/o3 和 DeepSeek R 系列正在定义"推理模型"新品类。

### 3.4 AI Agent 原生
月之暗面 Kimi Claw、智谱 Agent 框架、DeepSeek tool use 能力——模型从对话工具演化为能自主调用工具、执行任务的 Agent 操作系统。

## 四、PE/VC 投资机会

### 🟢 最优：垂直行业模型 + 开源商业化

**逻辑：** 基础模型竞争太激烈且烧钱 > 垂直行业有壁垒（数据+行业知识+监管），可形成数据飞轮。开源模型商业化（企业服务版）是对标 Red Hat 的 AI 版本。

**方向：**
- **AI+医疗** — 百川 All-in 医疗，联影/推想影像 AI，百亿级市场，壁垒最高
- **AI+金融** — 澜舟科技（孟子模型），智能投研、合规、风控，金融机构付费意愿强
- **AI+法律** — 幂律智能/秘塔 AI，合同审查+法律检索，数十亿市场
- **AI+能源** — 见 [AI应用层](AI应用层.md)，功率预测、电力交易、VPP

### 🟡 谨慎：Agent 平台层

模型层提供 Agent 能力的"大脑"，但 Agent 本身更多是应用层机会。参见 [AI应用层](AI应用层.md)。

### 🔴 避免：基础大模型独立投资

- 估值过高（100-500x PS）
- 单次训练成本 $数千万-数亿
- 极低毛利率（API 价格战）
- 退出不确定（港股流动性有限）
- 已转为巨头/国家队游戏

> **一级视角：** 基础模型已不适合独立财务投资。2026 年确定性最高的策略是**找垂直行业里最能形成"数据飞轮"的模型公司**——模型越用越好→客户越多→数据越多→模型壁垒越高。其次是**押注开源模型生态的平台层**（如模型托管、微调服务、推理加速引擎）。

## 五、关键风险

| 风险 | 说明 |
|------|------|
| 技术路线颠覆 | Transformer 被突破（Mamba/RWKV/状态空间模型）→ 现有架构投资价值清零 |
| 估值泡沫破裂 | 万亿 RMB 市值对应不到 50 亿收入，回调风险极大 |
| 开源商品化 | DeepSeek / Llama 开源持续低成本，闭源模型溢价能力下降 |
| 人才竞争激烈 | 顶级 AI 科学家年薪千万级，初创无法与大厂和 OpenAI 竞争 |
| 综合监管 | 生成内容合规/数据安全/算法备案，增加成本 |
