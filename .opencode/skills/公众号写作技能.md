# 公众号写作技能

当我加载此技能时，我会按以下流程协助公众号文章创作。

## 输入要求

- 主题/选题方向
- 目标读者（投资人群体 / 行业从业者 / 大众）
- 字数要求（默认 2000-3000 字）
- 参考风格文章（如有）
- 核心观点或数据（如有）

## 创作流程

### 1. 选题与定位

确认：
- 文章的 core message（用一句话说清楚这篇文章想让读者带走什么）
- 切入点（从什么角度切入）
- 差异化（和同类文章相比独特之处）

### 2. 结构生成

```
一、开头（Hook）
   - 用数据/事件/问题引入
   - 快速建立读者共鸣
   
二、主体（论证）
   - 2-4 个核心论点
   - 每个论点：事实/数据 → 分析 → 投资含义
   - 穿插案例/对比/图表
   
三、结尾（Call to Action / 升华）
   - 总结核心观点
   - 给出明确的投资/思考建议
   - 引导互动（提问/讨论）
```

### 3. 写作风格

参考要素：
- 标题：信息密度高 + 有观点倾向
- 开篇：前 3 句必须抓住注意力
- 段落：短段落为主（手机阅读优化）
- 数据：关键数据突出，标注来源
- 语气：专业但不晦涩，有立场但不情绪化
- 金句：每 500 字左右设置一个可被引用的金句

### 4. 标题优化

提供 3-5 个备选标题方案，标注推荐理由。

### 5. 最终交付

- 完整文章（Markdown 格式，可用 wenyan-mcp 直接推送到公众号草稿箱）
- 配图建议（关键数据可视化 / 信息图 / 封面图方向）
- 摘要/导语（公众号封面下方导读文字）

## 迭代学习

- 你提供的过往文章 → 我学习风格偏好
- 阅读量/互动数据反馈 → 我优化选题和标题策略
- 你的修改习惯 → 我减少需要修改的部分
