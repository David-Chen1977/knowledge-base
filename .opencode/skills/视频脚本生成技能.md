# 视频脚本生成技能

当我加载此技能时，我会按以下流程将文章转化为视频脚本。

## 依赖工具

- `video_dub.py` — 配音生成（在 `/Volumes/移动硬盘/工具包与模板/内容管线/`）
- `edge-tts` — 微软 AI 语音（已安装）
- MoneyPrinterTurbo — 视频合成（在 `/Volumes/移动硬盘/MoneyPrinterTurbo/`）

## 输入要求

- 原始文章（网站版或公众号版）
- 视频时长目标（默认 5-15 分钟）
- 目标平台（视频号/B站）
- 参考风格（如有参考视频链接）

## 创作流程

### 1. 文章→口播稿转换

从网站深度文章提取核心内容，转为口语化脚本：

```
文章 3000字 → 口播稿 1500-2000字（5-8分钟）
```

**转换规则：**
- 长篇背景铺陈 → 前 3 句直击重点
- 专业术语 → 保留 + 简短解释
- 复杂句式 → 拆成短句（每句 ≤ 25 字）
- 数据保留 → 但用"三倍"、"一半"替代精确数字
- 添加口语连接词："说白了"、"那问题来了"、"你猜怎么着"
- 每段 ≤ 2 句话（方便出图）

### 2. 脚本结构

```
00:00-00:15  开场（我是陈道雷，今天聊...）
00:15-01:00  一个矛盾/现象/故事（钩子）
01:00-03:00  核心分析（2-3 个观点）
03:00-04:30  数据和案例（充实论证）
04:30-05:00  总结 + 投资/思考建议
05:00-05:15  结尾 CTA（点赞关注转发）
```

### 3. 分镜与画面建议

每条脚本配套分镜说明：
- 口播（头像出镜 / AI 数字人）
- 图表/数据动画（与内容同步）
- 实景/素材画面（行业场景）
- 文字标题（核心观点放在屏幕上）

### 4. 配音生成

口播稿定稿后，用 `video_dub.py` 生成配音：

```
python3 video_dub.py text "口播稿全文" -o output.mp3 --voice zh-CN-XiaoxiaoNeural --rate +10%
```

### 5. 视频合成

配音准备就绪后：
- 通过 MoneyPrinterTurbo 或专业剪辑工具合成
- 字幕自动生成（ASS/SRT 格式）
- 封面设计（关键数据 + 标题）

## 输出格式

- 口播稿 Markdown（含时间轴和分镜）
- 配音文件 .mp3
- 字幕文件 .srt（可选）
- 封面设计说明（可选）
