## 三、各环节分析

### 3.1 电芯制造

- **LFP：** 规模效应显著，龙头成本已<0.3元/Wh，但产能过剩
- **钠电：** 2025-2026量产元年，成本比LFP高10-20%，2027有望平价
- **液流：** 大连融科/北京普能等已实现百MW级交付

**一级机会：** ⭐ 钠电（硬碳负极/电解液初创）、⭐ 液流（核心材料国产化）

### 3.2 BMS/EMS/PCS

- **PCS：** 竞争激烈，同质化，阳光电源/上能/科华/盛弘 CR5 >70%
- **BMS/EMS：** 软件定义差异化，AI+储能优化调度是趋势
- **温控：** 液冷取代风冷（3-5MWh+），液冷机组国产化

**一级机会：** ⭐⭐ EMS/储能数字化平台（AI调度+电力交易）

### 3.3 储能系统集成

- 价格战白热化（2025年系统报价已降至0.5-0.7元/Wh）
- 差异化方向：海外认证（UL9540/IEC62619）、安全设计、数字化运维

**一级机会：** ⭐ 新型储能集成商（差异化技术路线）

### 3.4 工商业储能

- 最活跃的一级赛道之一
- 商业模式：合同能源管理（EMC）为主，业主0投入分享收益
- 盈利逻辑：峰谷价差套利 + 需求侧响应补贴 + 容量管理
- 关键变量：分时电价政策、电池循环寿命

**一级机会：** ⭐⭐⭐ 工商业储能运营平台（渠道+数字化+运维网络）

## 七、关键趋势

1. **长时储能政策红利** — 4h+、8h+ 强制配储要求逐步出台
2. **独立储能商业模式跑通** — 现货市场+容量电价+辅助服务复合收益
3. **海外市场高毛利** — 欧美储能市场利润率是国内的2-3倍
4. **储能资产证券化** — REITs/ABS 退出通道打开
5. **AI + 储能** — 智能调度提升IRR 2-5个百分点
