## 🔑 2026年关键变化

1. **负电价正式化**：辽宁引入-100元/MWh下限
2. **容量电价分化**：广东气电涨164%-296%，煤电涨65%
3. **新能源全面入市**：136号文要求30-50%电量入市
4. **售电盈利模式重构**：广东取消浮动费用，批零价差消失

## 📌 投资含义

- **电价梯度 = 选址梯度**：蒙西/甘肃/陕西低价区→算电协同首选
- **地理套利空间**：蒙西(~240) vs 广东(~460)，价差近1倍
- **时点套利**：陕西日前最低66元 vs 最高313元，日内价差4.7倍
