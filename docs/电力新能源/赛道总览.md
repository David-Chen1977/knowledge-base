## 一、宏观背景

### 双碳目标时间线

| 时间节点 | 目标 |

| 2025 | 非化石能源消费占比 20% ✓ |

| 2060 | 碳中和 |

### 十五五规划重点（2026-2030）

- 新型电力系统：源网荷储一体化、柔性互联
- 新能源装机：风光年增 2亿千瓦+（已超额完成十五计划）
- 电力市场化改革：现货市场全覆盖、容量电价全面推行
- 氢能：纳入国家能源管理体系
- 碳市场：扩围至钢铁/水泥/电解铝

## 二、各子赛道概览

### 1. 光伏（新技术方向）

**成熟度：** 主产业链成熟，新技术早期
**PE/VC 机会：** Perovskite 叠层、BIPV、光伏回收、光伏+数字化
**当前周期：** 产能过剩出清期，制造业利润承压，技术创新溢价

### 2. 储能

**成熟度：** 高速成长期
**PE/VC 机会：** 钠离子电池、液流电池、工商业储能、长时储能
**当前周期：** 政策红利释放期，产能开始过剩，技术分化加剧

### 3. 风电（新技术方向）

**成熟度：** 主产业链成熟，深远海/浮式早期
**PE/VC 机会：** 浮式风电平台、智慧运维、风电叶片回收、风电数字化
**当前周期：** 陆上平价，海上降本关键期

### 4. 氢能

**成熟度：** 早期成长期
**PE/VC 机会：** 绿氢制备（PEM/AEM电解槽）、氢储运技术、燃料电池核心部件
**当前周期：** 政策驱动转向经济性驱动，绿氢降本是关键拐点

### 5. 智能电网/虚拟电厂

**成熟度：** 成长期
**PE/VC 机会：** VPP 聚合平台、电力交易 AI、微电网、需求侧响应
**当前周期：** 电改深化利好，数字化+市场化双轮驱动

### 6. 碳市场

**成熟度：** 启动期
**PE/VC 机会：** 碳资产管理平台、碳监测 CEMS、碳金融、ESG 数据
**当前周期：** 全国碳市场扩围在即，CCER 重启后供给不足

### 核心关注信号

1. **产能出清进度** — 制造业环节何时供需再平衡
2. **新型储能成本曲线** — 钠电/液流何时平价
3. **绿氢 LCOH** — 何时降至 20元/kg 以下
4. **电力现货市场进展** — 哪些省份率先全面运行
5. **碳价走势** — 从当前 70-100元/吨能否突破 200元

## 四、风险因素

| 风险 | 影响 |
|------|------|
| 产能过剩持续 | 制造业标的价值缩水，供应链利润挤出 |
| 政策节奏变动 | 补贴退坡速度、碳市场扩围时间不确定 |
| 技术路线替代 | 锂电→钠电→固态，钙钛矿→晶硅 |
| 地缘政治 | 光伏出海壁垒、风电出海认证、技术封锁 |
| 估值泡沫 | 部分赛道（储能/AI+能源）过热 |
