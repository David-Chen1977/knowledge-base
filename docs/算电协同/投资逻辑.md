# 算电协同 · 投资逻辑框架

**视角：** PE/VC 一级股权投资
**当前阶段：** 早期布局期（2025-2027 窗口期）

---

## 一、三句话判断

> **第一句："双 80"是硬约束不是口号——绿电占比 80% + PUE 1.25，决定了谁的节能体系好谁才能获批新建。**
> 能效从"可选项"变为"准入门槛"。

> **第二句：看得见的钱在设备（液冷/储能），看不见的钱在平台（调度/交易/能效优化）。**
> 一级市场应该赌平台型机会，设备是二级市场的。

> **第三句：2026年5月广东数据中心 VPP 现货交易首单证伪了"数据中心不能参与电力市场"的假设。**
> 最大机会在帮数据中心省电和赚钱的轻资产平台——AI 制冷优化 SaaS 和 VPP 聚合平台。

## 三、投资策略建议

### 3.1 优先布局（P0）

1. **AI制冷优化SaaS** ⭐⭐⭐⭐⭐ — 全球空白，轻资产高壁垒
   - **投资逻辑**：海外 Phaidra 已为 Google 降低 30% 制冷能耗，国内对标空白
   - 数据中心制冷占能耗 30-40%，AI 优化可节省 10-30%
   - 毛利率 60-80%，SaaS 模式可复制
   - **关键验证**：算法精度、跨平台兼容性（能否适配多种 GPU/制冷方案）
   - **国内现状**：暂无专注 AI 制冷优化的独立公司

2. **AI电力交易平台** — 壁垒最高、资本效率最高
   - 投资逻辑：电力现货市场扩容→交易频次增加→AI报价策略需求爆发
   - 关键验证：是否有真实交易数据、策略夏普比率
   - 代表：瓦特智能、清大科越

3. **算力调度平台** — 网络效应，先发优势明显
   - 投资逻辑：跨区域/跨云调度需求随算力增长指数上升
   - 关键验证：签约算力规模、客户留存率
   - 代表：算力互联、星链算力

### 3.2 重点跟踪（P1）

4. **数据中心需求响应平台** — 政策驱动确定性强（2026年5月国内首单验证）
5. **VPP聚合数据中心平台** — AI+能源交叉，增长快

### 3.3 关注等待（P2）

5. **浸没式液冷** — 技术路线未定，等标杆案例
6. **源网荷储一体化项目** — 重资产，适合产业资本而非财务投资

## 七、投资方角色定位（商业模式回答）

> **回答待深化问题 P0 #1"谁最终付费？"—— 终端付费方始终是算力用户，投资方作为总集成商切入。**

### 7.1 价值链条

```
绿电供给方                      算力用户（最终付费）
    ↑                                ↑
    |         ┌──────────────┐        |
    └─────────┤  总集成商    ├────────┘
              │  （我们）    │
    ┌─────────┤             │◄───────┐
    ↑         └──────────────┘        ↑
 绿电交易方                       设备厂商
（代理分成）                     （代理分成）
```

### 7.2 四种切入角色（从轻到重）

| 角色 | 模式 | 资本投入 | 利润空间 | 核心能力要求 |
|------|------|---------|---------|------------|
| **A. 纯出资方** | 投钱给算力出租方/总集方，吃资本回报 | 高 | 中等（IRR 8-12%） | 募资能力、项目判断 |
| **B. 渠道代理方** | 联合设备方（液冷/储能/GPU）拓展订单，按成交额抽成 | 低 | 中高（佣金3-8%） | 客户资源、行业关系 |
| **C. 绿电撮合方** | 连接绿电项目与算力需求方，吃绿电交易/PPA撮合佣金 | 低 | 中（佣金2-5%） | 绿电资源、电力交易理解 |
| **D. 总集成方** | 整合节能技术+绿电+设备+运营，给算力出租方交钥匙 | 中 | 高（净利15-25%） | 技术理解、供应链、交付能力 |

### 7.3 策略建议：分层组合

不单选一种角色，而是**根据项目情况灵活组合**：

- **核心竞争力 → 做D（总集成商）**：当团队/资源到位，吃最大利润
- **资源变现 → 做B+C**：有客户关系做渠道，有绿电资源做撮合
- **能力不足 → 做A**：纯出资，借助别人的专业能力
- **理想路径**：A（建立信任）→ B+C（积累资源）→ D（能力成熟后收割）

### 7.4 落地示例

假设一个算力出租方需要建千P智算中心：

```
算力出租方 → 需要：设备采购 + 液冷方案 + 绿电供应 + 运营服务
                    ↓
我们作为总集成商 → 整合：
  ├── 设备采购（联合英维克/高澜 → 赚代理佣金）
  ├── 绿电供应（对接风光电站 → 赚绿电撮合佣金）
  ├── 节能方案（引入AI制冷SaaS → 赚技术整合费）
  └── 自己出资一部分（赚资本回报）
```
