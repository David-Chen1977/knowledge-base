# 算电协同——数据中心与新能源融合的新商业模式

## 摘要

算电协同是 2026 年新兴的投资主题。国家六部门 2025.12 发布《关于促进企业数据资源开发利用的意见》，发改委等部门 2026.3 发布《关于促进可再生能源绿色电力证书市场高质量发展的意见》，为算电协同奠定政策基础。核心逻辑：数据中心作为柔性负载，与新能源电站共建，实现绿电直供+余电上网+算力调度三位一体。典型项目 IRR 预计 12%-18%，中西部（内蒙/宁夏/甘肃）度电成本 0.20-0.30 元/kWh，竞争优势显著。

## 政策基础

| 政策 | 发布单位 | 要点 |
|------|---------|------|
| 数据产业顶层规划 (2026) | 六部门 | 鼓励数据产业与能源产业协同发展 |
| 绿证市场高质量发展意见 (2026.3) | 发改委等部门 | 完善绿证制度，降低绿电交易门槛 |
| 数据中心绿色低碳发展行动方案 (2025) | 发改委 | 数据中心可再生能源利用率目标 80%+ |
| 全国一体化大数据中心体系 (2024+) | 发改委 | 东数西算，算力西部布局 |

## 商业模式

### 核心结构

```
新能源电站 (风电/光伏)
    ↓ 绿电直供
数据中心 (柔性负载)
    ↓ 余电上网
电网/交易市场
```

### 三层价值

1. **绿电直供**：数据中心直接消纳新能源电力，降低电费 + 满足可再生能源利用率要求（80%+）
2. **余电上网**：多余绿电/绿证上网销售，获取额外收益
3. **算力调度**：利用数据中心作为柔性负载，实现需求侧响应，参与辅助服务市场

### 盈利模式

| 收益来源 | 占比 | 说明 |
|---------|------|------|
| 算力服务 | 60%-75% | 主要收入来源 |
| 绿电节约 | 15%-25% | 较市电价低 30%-50% |
| 余电上网 | 5%-10% | 多余电力上网销售 |
| 辅助服务 | 3%-5% | 需求侧响应等 |

## 经济性分析

### 中西部数据中心 vs 东部数据中心

| 维度 | 中西部（内蒙/宁夏/甘肃） | 东部（北京/上海/广东） |
|------|----------------------|---------------------|
| 度电成本 | 0.20-0.30 元/kWh | 0.50-0.80 元/kWh |
| 电价优势 | 省 50%+ | - |
| 土地成本 | 1/10 | 基准 |
| 网络延迟 | 10-30ms | <5ms |
| 适用场景 | 训练/推理/存储 | 实时/推理 |

### 典型 IRR

| 项目类型 | IRR 区间 | 说明 |
|---------|---------|------|
| 纯数据中心 | 8%-12% | 无绿电配套 |
| 数据中心 + 绿电 (中西部) | 12%-18% | 绿电直供降电费 |
| 数据中心 + 绿电 + 调度 (东部) | 10%-15% | 灵活性收益 |
| 算电协同园区 (中西部) | 15%-20% | 规模化+调度优化 |

## 标杆项目

- **内蒙古乌兰察布**：阿里/华为数据中心与风电/光伏共建
- **宁夏中卫**：西部云基地 + 光伏电站
- **甘肃庆阳**：东数西算节点 + 新能源基地
- **张家口**：数据中心 + 风电直供（可再生能源利用率已达 67%+）

## 投资逻辑

1. **中西部是主战场**：电价优势不可替代，东数西算加速布局
2. **算力需求爆发**：AI 大模型训练/推理驱动算力需求 CAGR 50%+
3. **政策强制可再生能源利用率**：80%+ 目标倒逼数据中心配建绿电
4. **柔性调度价值释放**：数据中心参与辅助服务市场增加收益

## 风险提示

1. 网络延迟瓶颈限制实时推理场景
2. 新能源出力波动性需配套储能
3. 跨省绿电交易机制仍在完善
4. 数据中心 PUE 能耗指标监管趋严
5. 算力芯片供应不确定性
