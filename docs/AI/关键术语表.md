# AI · 关键术语表

**用途：** PE/VC 一级市场投资研究参考

---

## 一、基础概念

| 术语 | 说明 |
|------|------|
| **大模型 / LLM** | Large Language Model，大规模语言模型，如 GPT-4、GLM-4 |
| **多模态** | Multimodal，同时处理文本+图像+音频+视频的模型能力 |
| **AGI** | Artificial General Intelligence，通用人工智能 |
| **AI Agent** | 能自主感知、规划、执行任务的 AI 系统 |
| **RAG** | Retrieval-Augmented Generation，检索增强生成，外挂知识库 |
| **MoE** | Mixture of Experts，混合专家模型，提高参数效率 |
| **Transformer** | 基础模型架构，Attention is All You Need |
| **Diffusion** | 扩散模型，图像/视频生成核心架构（Stable Diffusion/Sora） |
| **RLHF** | Reinforcement Learning from Human Feedback，人类反馈强化学习 |
| **SFT** | Supervised Fine-Tuning，监督微调 |
| **Prompt Engineering** | 提示词工程，优化输入以获得更好的模型输出 |
| **温度 (Temperature)** | 控制生成随机性的参数，越低越确定 |
| **幻觉 (Hallucination)** | 模型生成看似合理但实际错误的内容 |

## 二、模型训练/优化

| 术语 | 说明 |
|------|------|
| **预训练** | Pre-training，在海量数据上训练基础模型 |
| **微调** | Fine-tuning，在特定领域数据上调整模型 |
| **LoRA** | Low-Rank Adaptation，高效微调方法，修改少量参数 |
| **量化** | Quantization，降低模型精度（FP16→INT8/INT4）以减小体积 |
| **蒸馏** | Knowledge Distillation，大模型教小模型 |
| **剪枝** | Pruning，删除不重要的模型参数 |
| **上下文窗口** | Context Window，模型能处理的输入长度（Token数） |
| **注意力机制** | Attention，Transformer 核心，让模型关注输入的不同部分 |
| **Scaling Law** | 模型性能随参数量/数据量/算力增长的可预测规律 |

## 三、基础设施

| 术语 | 说明 |
|------|------|
| **GPU** | Graphics Processing Unit，AI 训练/推理主力芯片 |
| **TPU** | Tensor Processing Unit，Google 专为 TensorFlow 设计 |
| **NPU** | Neural Processing Unit，神经网络加速器 |
| **HBM** | High Bandwidth Memory，高带宽显存（HBM2e/HBM3/HBM4） |
| **CoWoS** | Chip-on-Wafer-on-Substrate，台积电 2.5D 先进封装 |
| **CXL** | Compute Express Link，高速缓存一致性互联协议 |
| **存算一体** | Processing-in-Memory (PIM)，在存储中计算，减少数据搬运 |
| **算力租赁** | 按需租用 GPU/算力，如 CoreWeave/Vast.ai |
| **FLOPS** | FLoating-point Operations Per Second，算力单位 |
| **TOPS** | Trillion Operations Per Second，常用于端侧AI芯片 |
| **TDP** | Thermal Design Power，功耗散热设计 |
| **智算中心** | AI 专用数据中心，部署大量 GPU/NPU |

## 四、生成式 AI

| 术语 | 说明 |
|------|------|
| **AIGC** | AI Generated Content，AI 生成内容 |
| **文生图** | Text-to-Image，如 Midjourney/DALL-E/Stable Diffusion |
| **文生视频** | Text-to-Video，如 Sora/可灵/Emu Video |
| **文生音乐** | Text-to-Music，如 Suno/Udio |
| **数字人** | Digital Human，AI 驱动的虚拟人形象 |
| **文生3D** | Text-to-3D，如 Meshy/Luma Genie |
| **ControlNet** | 精确控制图像生成姿势/构图的技术 |
| **Lora（不是LoRA）** | 分布式个性化微调模型文件 |

## 五、AI 应用/行业

| 术语 | 说明 |
|------|------|
| **Copilot** | AI 编程/办公助手模式 |
| **RPA + AI** | AI 增强的机器人流程自动化 |
| **AI 辅助诊断** | AI 辅助医学影像/病理/心电图诊断 |
| **智能投研** | AI 辅助投资研究/数据分析 |
| **合成数据** | AI 生成的训练数据（解决真实数据不足） |

| **MLOps** | Machine Learning Operations，模型部署运维流水线 |
| **AI 安全** | AI 对齐/红队测试/对抗攻击防御 |
| **可解释性 AI** | XAI，理解模型决策原因的技术 |
