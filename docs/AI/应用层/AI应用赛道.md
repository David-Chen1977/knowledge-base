## 二、AI Agent 平台（2026 最大热点）

| 方向 | 说明 | 代表 |
|------|------|------|
| 通用AI Agent | 自主规划执行任务 | Manus、AutoGPT |
| 垂直AI Agent | 专业领域自主工作流 | Devin(编程)、法律Agent |
| Agent 平台/框架 | 构建Agent的开发平台 | LangChain、Dify、Coze |
| Agent 托管/运行 | Agent 云服务 | — |

**关注点：**
- Agent 的"可靠性"壁垒——用户容忍度极低
- 工具调用准确率是关键瓶颈
- 长期看 Agent 平台 > 单个 Agent 产品
