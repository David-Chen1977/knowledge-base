# AI Agent 在电力调度中的应用——从演示品到省级部署

## 摘要

AI Agent 在电力调度领域已从演示品走向省级规模化部署，并非 PPT 产品。广东电网"明月"AI调度员日均处置 100+ 告警信号、拟写 50+ 指令。广西电网"阿度"断面调控智能体秒级感知。长治 AI 机器人将核心业务耗时缩短 80%+。南网"大瓦特"落地 700+ 实景场景。多家已通过知识图谱+强化学习+微调解决"幻觉"问题。主要仍在电网调度侧，市场化交易侧 AI Agent 仍处早期。

## 核心案例

### 广东电网"明月"AI调度员

- 国内首个多智能体协作省级电网 AI 调度员
- 基于南网"大瓦特-天璇"模型底座
- 融合知识图谱 + 强化学习 + 大模型微调技术，规避大模型"幻觉"
- 日均能力：编制 10+ 调度报表、拟写 50+ 调度指令、处置 100+ 告警信号
- 智能话务通知 200+ 次/日，全时段监护安全与市场合规 1000+ 项
- 全链条应急处置时间从小时级缩短至 5 分钟内
- 工作模式从"一对一"变成"多对多"

### 广西电网"阿度"断面调控智能体

- 基于"大瓦特-天璇"大模型 + 强化学习（GRPO）算法
- 秒级感知电网断面状态，智能预测与自主调控
- 2024 年底上线，显著提升正常态与极端故障场景控制及时性
- 大小模型融合方案（大模型调度、小模型执行分析）
- 解决调度监控中海量告警信号辨识难题

### 长治电网 AI 调度机器人

- 国网系统部署，搭载"1+3+5"智能体系
- 核心业务耗时缩短 80% 以上
- 电网风险扫描覆盖率 100%，周期从 30 分钟 → 分钟级
- 重要用户供电路径分析从十几分钟 → 秒级
- 五大场景：电网运行感知、设备运维、供电形势评估、风险预控、重要用户保障

### 深圳供电局配电网监视 Agent

- CPU+GPU 异构 + 大模型融合
- 首创"大模型调度、小模型执行分析"联动模式
- 应用于配电网监视 & 操作票审核
- 百秒零差错完成 N-1 安全校核

### 南方电网"大瓦特"大模型

- 能源行业首个全栈国产化企业级 AI 原生智能执行助手
- 落地 700+ 实景场景（调度优化、设备巡检、智能客服等）
- 替代超 80% 人工巡检工作
- 常规任务处理耗时平均降低 80%
- 开箱预置 50+ MCP 和 Skills

## AI 幻觉问题

多家已实质解决：

| 方案 | 技术手段 | 效果 |
|------|---------|------|
| "明月" | 知识图谱+强化学习+大模型微调 | 规避"幻觉" |
| "阿度" | GRPO 强化学习算法 | 精准推理，可解释性 |
| 为恒 WHES OS | AI 策略解释与归因回放 | 首创"黑盒"解释功能 |
| 行业趋势 | 大小模型融合 | 实测验证替代理论推测 |

## 商业化进展

### 为恒智能 WHES OS

- 行业首款 AI-native 原生智能化能源操作系统
- AI Agent 通过自然语言理解用户需求，自动生成能源策略
- 实测：项目年均收益提升 8%-15%，调度成本下降 20%
- 15 分钟级实时滚动优化
- 覆盖工厂峰谷套利、光储充、微电网、VPP 等场景
- 2026 年 2 月向港交所递交上市材料

### 结论

| 维度 | 评价 |
|------|------|
| 技术成熟度 | 已从演示品走向省级 7x24 生产部署 |
| 误报率 | 核心难题已解决（知识图谱+强化学习），未公布精确误报率数据 |
| 主要形态 | 电网调度侧（与调度员协作），非替代调度员 |
| 市场化交易侧 | 仍处早期，AI Agent 参与现货市场交易决策的案例较少 |
| 储能运营侧 | 为恒智能等企业已推出 AI Agent 商业化产品 |

## 风险提示

1. 精确误报率数据未公开披露，难以横向对比
2. 当前以辅助调度员为主，全自主决策仍有距离
3. 市场化交易侧 AI Agent 成熟度不足
4. 行业标准缺失，各家方案差异大，互操作性有限
5. 对电网安全的高要求意味着 AI Agent 的部署审核周期长
