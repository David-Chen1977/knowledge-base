## 🔄 自更新机制

### 知识库 ↔ 内容日历 联动规则

```
知识库层                             内容日历层
─────────────────────────────────────────────
当前关注.md P0 设为"算电协同"    →    算电协同选题升至 P0
                                      （如已产出则降为复盘）
                                      
待深化问题.md 新增 P0 问题       →    自动生成选题建议
                                      （如"谁最终付费？"→"算电协同商业模式"选题）
                                      
知识库新增赛道文件               →    加入待定选题池
                                      
知识库某文件长时间未产出内容      →    提示"可转化为文章"
```

### 触发时机

| 触发条件 | 动作 | 自动/手动 |
|---------|------|----------|
| `当前关注.md` P0 变化 | 刷新选题优先级排序 | 🔄 自动 |
| `待深化问题.md` 新增 P0 | 生成选题建议 | 🔄 自动 |
| 内容产出完成（任何一期） | 更新进度表 + 知识库回流 | 🔄 自动 |
| 每周日 | 检查下周排期，提示准备素材 | 📋 提醒 |
| 每月1日 | 季度排期回顾+调整 | 📋 提醒 |

## 内容格式标准

### 公众号文章
- **长度：** 1000-1500 字
- **格式：** Markdown → 微信编辑器
- **风格：** 精简版，开篇 2-3 句抓人，段落短
- **结尾：** CTA 引导 + 署名 + 日期

### 网站深度文章
- **长度：** 2000-4000 字
- **格式：** Astro `.astro` 文件（chendaolei-website）
- **风格：** 完整分析，数据密集，有框架归纳

### PPT
- **长度：** 15-25 页
- **格式：** PowerPoint
- **内容：** 核心观点 + 关键数据 + 投资图谱

### 视频
- **长度：** 5-15 分钟
- **格式：** 稿子 → 录制
- **内容：** 口语化的深度解读
