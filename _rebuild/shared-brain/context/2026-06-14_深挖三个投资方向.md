## 方向一：算电协同 — "绿色瓦特"支撑"规模比特"

### 2. 政策框架 — 2026年关键进展

**国家能源局 2026.05.15 发文《以算电协同赋能新型能源体系》**，提出"三协同"体系：

- **时空协同**: 东数西算节点 ↔ "沙戈荒"新能源大基地深度融合；跨省跨区绿电交易
- **技术协同**: 智能调度 + 柔性负荷调控 + 精准预测 + 高效储能
- **机制协同**: 算力-电力价格联动 + 绿电认证与碳减排互认 + 算力券/绿电补贴

**PUE 硬约束**: 新建大型 DC PUE ≤ 1.15 (2026年生效)，液冷是达标刚需手段

### 3. 落地模式

| 模式 | 代表地区 | 技术方案 |
|------|---------|---------|
| **绿电直连** | 内蒙古乌兰察布·中金数据 | 新能源电站→直供电→算力中心，100%绿电可溯源 |
| **绿电聚合供应** | 甘肃庆阳/河北张家口/宁夏中卫 | 源网荷储一体化，多个新能源电站聚合供应DC集群 |
| **跨区电算联合调度** | 上海电力×中国电信 (试点) | DC虚拟电厂参与辅助服务，算力迁移"先知电量、后调算力" |
| **分布式绿电自建** | 腾讯仪征 (屋顶光伏1.3万kW) | 屋顶光伏+储能，年均消纳1210万kWh |

### 4. 八大枢纽节点政策矩阵

| 枢纽 | 差异化定位 |
|------|-----------|
| **内蒙古** | 绿电直供 + 电价优惠，适合高时延批处理任务 |
| **宁夏** | "东数西算"节点，光资源好，储能配置 |
| **甘肃** | 现货电价最低(~145元/MWh)，风光资源好 |
| **河北(张北)** | 靠近首都，秦淮数据环首都·太行山基地，100%绿电 |
| **长三角** | 低时延网络，跨境算力支持 |
| **粤港澳** | 低时延+高电价(637元/MWh)，边缘计算 |
| **成渝** | 水电丰富，绿电比例高 |
| **贵州** | 气候凉爽，天然冷却优势 |

### 5. 全球对标

| 地区 | 模式 | 特点 |
|------|------|------|
| 中国 | **国家统筹+基建先行** | 东数西算，算电协同上升为国家战略 |
| 美国 | **市场主导+能源争夺** | 依赖气电过渡+长期锚定核电，绕过公共电网直供 |
| 欧盟 | **能效优先+绿色合规** | 强制碳足迹披露，限制高耗能扩张 |
| 日本 | **瓦特-比特协同** | 从东京/大阪向北海道等可再生能源富集地区分散 |

**中国**: 数据港、光环新网、奥飞数据、润泽科技、万国数据(GDS)、世纪互联(VNET)、秦淮数据

### 2. ⭐ 钠离子电池 — 2026年最大结构性机会

#### 2.3 成本与性能曲线

```
当前状态 (2026H1):
  LFP 电芯: ~0.38 元/Wh
  钠电电芯: ~0.47 元/Wh (头部企业)

关键拐点:
  若碳酸锂 > 20万元/吨 + 钠电规模上量 → 2026年底成本打平
  头部企业乐观预测: 2026年电芯成本可低至 0.35 元/Wh
  2027年: 全面平价放量时代

性能:
  - 循环寿命: 6000-15000次 (CATL储能专用)
  - 低温性能: -40℃保持率 > 90% (锂电~60%)
  - 安全性: 不起火不爆炸，本质安全
```

#### 2.4 ⭐ 数据中心储能专用场景

**钠电在AIDC储能场景具有天然匹配优势**：

| 需求 | 钠电优势 |
|------|---------|
| 低温稳定 (西北/东北DC) | -40℃保持90%电量 |
| 长循环寿命 (全生命周期成本) | 15000次 |
| 安全性 (不允许起火) | 本质安全，不燃 |
| 资源自主 (不受锂约束) | 钠储量是锂的数百倍 |

**宁德时代**已在ESIE2026展会上正式推出**储能专用钠离子电池** (300+Ah, 15000次循环)，直接对准AIDC储能场景。

## 综合分析：三条赛道的关联性

```
                ┌──────────────────┐
                │  AI 算力爆发      │
                │  (电力需求暴增)    │
                └────────┬─────────┘
                         │
          ┌──────────────┼──────────────┐
          ▼              ▼              ▼
   ┌────────────┐ ┌────────────┐ ┌────────────┐
   │ 算电协同    │ │ 长时储能    │ │ 液冷系统    │
   │ (绿电直供)  │ │ (消纳+调峰) │ │ (散热必需)  │
   └────────────┘ └────────────┘ └────────────┘
          │              │              │
          └──────────────┼──────────────┘
                         ▼
               ┌──────────────────┐
               │  DC 绿电+储能+液冷   │
               │  一体化解决方案      │
               └──────────────────┘
```

**三条赛道高度联动，共同服务于AI算力的能源基础设施需求：**

1. **算电协同**解决"电从哪来"——绿电+政策保障
2. **长时储能**解决"怎么存"——消纳波动+备电安全
3. **液冷系统**解决"怎么散"——高密度散热刚需

**当前确定性排序**: 液冷系统 > 钠电储能 > 算电协同
- 液冷: 需求最刚性，2026年放量确定性最高
- 钠电: 2026年量产元年，拐点已至但需跟踪产能兑现
- 算电协同: 政策驱动，落地节奏取决于电网改革进度
