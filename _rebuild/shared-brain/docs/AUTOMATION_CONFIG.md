## 故障排查

### 自动化未触发
1. 检查调度时间是否正确
2. 检查工作目录是否存在
3. 查看自动化运行历史

### 脚本执行失败
1. 检查 Node.js 是否可用
2. 检查脚本路径是否正确
3. 手动运行脚本测试
