## 三、四大工作流 · 人机协同方案

### 工作流 1：协议 Word 文本

```
你 → 提供协议初稿 / 修改要求
我 → 条款结构化分析、风险标注、修改建议
你 → 确认修改方向
我 → 生成修订版（含 tracked changes）
你 → 终审 → 发送法务
```

**我能做的：**
- 条款逐条解析（终止条款、赔偿责任、IP 归属、管辖法律）
- 风险点标注（红/黄/绿三级预警）
- 版本对比（前后差异高亮）
- 合规检查清单自动生成

**你需要提供的：**
- 风险偏好框架（哪些条款是红线，哪些可协商）
- 过往已签署协议样本（让我学习你的风格）
- 每次修改的反馈（帮助我迭代）

### 工作流 2：行业分析

```
你 → "帮我看看最近储能赛道有什么变化"
我 → 并行搜索 → 提取关键信息 → 输出结构化分析
你 → 补充问询 / 调整方向
我 → 深度挖掘 → 生成完整分析报告
```

**我能做的：**
- 多源信息并行聚合（新闻 + 研报 + 公告 + 论文）
- 竞争格局图谱绘制
- 关键指标数据提取与趋势分析
- 分析报告结构化输出（Word/PDF）

**你需要提供的：**
- 关注的企业/赛道清单
- 分析框架偏好（波特五力？PEST？价值链？）
- 信息源偏好（哪些研报平台、数据来源可信）

### 工作流 3：基金路演 PPT

```
你 → "帮我做一个新一期新能源基金的募资路演ppt框架"
我 → 生成完整大纲 + 每页关键内容 + 数据建议
你 → 调整结构和重点
我 → 用 pptx-mcp-server 生成 .pptx 文件
你 → 在 PowerPoint 中精修设计
```

**我能做的：**
- 完整 Deck 结构生成（封面→目录→市场分析→策略→业绩→团队→条款→QA）
- 每页标题 + 关键 message + 数据图表建议
- KPI 指标卡片自动排版
- 使用 IR 主题（投资者关系风格）直接生成
- 过往业绩数据可视化

**你需要提供的：**
- 基金基本信息（策略、规模、业绩、团队）
- 对标竞品材料（让我学习行业标准）
- 品牌视觉规范（logo、配色、字体）

### 工作流 4：自媒体输出（公众号 + 短视频）

```
你 → "最近光伏板块波动大，写一篇分析"
我 → 搜索最新行情/事件 → 草拟文章
你 → 调整语气/角度 → 确认
我 → 格式化为公众号风格 → 通过 wenyan-mcp 推送草稿箱
     ↓
我 → 提取核心观点 → 生成短视频脚本（60s / 3min）
你 → 拍摄 / AI 生成视频 → 发布
```

**我能做的：**
- 公众号长文：行业分析、投资思考、策略解读
- 短视频脚本：口播稿 + 分镜 + 字幕建议 + 画面描述
- 多平台适配：同一内容 → 公众号深度版 / 视频号精简版 / 小红书图文版
- 标题优化（提高打开率）
- 定期内容日历规划

**你需要提供的：**
- 个人风格样本（过往文章 3-5 篇，让我学习你的语言风格）
- 定位方向（偏深度分析？偏热点解读？偏投资理念？）
- 短视频参考（喜欢的博主风格、节奏）
