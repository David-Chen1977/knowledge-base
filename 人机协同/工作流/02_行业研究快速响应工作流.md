# 行业研究快速响应工作流

> 适用于：用户提出"帮我看看X赛道有什么变化"类快速研究

---

## 入口条件

- [ ] 用户给出赛道/公司名称
- [ ] 至少2-3句背景说明
- [ ] 明确研究目的（投资决策 / 跟踪研究 / 竞品对标）

## 执行流程

```
阶段一：信息收集（并行）
├── websearch 搜索最新行业动态
├── MCP 工具拉取数据（SEC/Yahoo Finance/China Energy）
├── 知识库已有材料检索
└── 时间窗口: 5分钟

阶段二：结构化分析
├── 核心结论（300字）
├── 市场规模与增速
├── 竞争格局图谱
├── 关键企业财务/进展
└── 时间窗口: 5分钟

阶段三：产出交付
├── Markdown 结构化报告 → 知识库归档
├── 发现值得深挖的方向 → 标记"可扩展三件套"
└── 更新 state.json
```

## 并行策略

```
同步执行:
  websearch("赛道+2026 最新动态")
  websearch("赛道+竞争格局")
  websearch("赛道+政策")
  MCP 财务数据拉取（如果是上市公司）

汇总后:
  → 写出结构化报告
  → 建议是否扩展为三件套
```

## 质量门禁

- [ ] 有明确的投资视角（PE/VC 视角，非二级炒股）
- [ ] 核心观点用一句话可说清楚
- [ ] 附数据来源/引用
- [ ] 文中不出现"根据公开资料"等模糊表述
