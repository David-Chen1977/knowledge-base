# 「一写多发」Prompt Template

## 用法

写好深度文章后，把全文粘贴到下面的 `{文章内容}` 处，发给 Claude。

## Prompt

```
你是一位资深财经/能源领域内容运营。请将以下深度文章改编成5个平台版本。

原文：
{文章内容}

---

### 要求

#### 1. 公众号版
- 文件名：`公众号版.md`
- 标题自拟（吸引人但不标题党）
- 1500-3000字，可带章节小标题
- 作者：陈道雷
- 语气：专业但有温度，适合行业人群转发
- 格式：纯 Markdown，无需考虑图片排版

#### 2. 知乎版
- 文件名：`知乎版.md`
- 标题多用"为什么/如何看待/深度分析"句式
- 开头有钩子，中间有数据支撑，结尾有观点
- 语气：比公众号更深入，像在回答一个专业问题
- 可带 `##` 小标题

#### 3. 头条版
- 文件名：`头条版.md`
- 标题要有数字/反差/悬念
- 每段不超过3行，手机阅读优化
- 小标题可以加 **加粗** 但不用 ##
- 不用 --- 分隔线，头条不识别
- 结尾引导评论/关注

#### 4. 网站深度版
- 文件名：`网站深度版.md`
- 带 frontmatter（title/date/author/description）
- 2500-4000字，六个以上的章节
- 语气：最完整的版本，SEO优化
- 可带 > 引用、--- 分隔线

#### 5. 短视频脚本 × 3条
- 文件名：`短视频脚本.md`
- 每条45-60秒
- 每条开头3秒必须有钩子
- 标注每段的视觉建议（配什么画面）
- 其中1条做数据冲击型，1条做反常识型，1条做政策翻译型

### 输出格式

输出为 Markdown，每条用 `---` 分割，顶部标明平台和文件名。
```

## 执行流程

1. 写好深度文章 → 保存到 `内容输出/{主题}/网站深度版.md`
2. 把全文复制到上面 Prompt 的 `{文章内容}` 处
3. Claude 输出全部5个版本
4. 逐一保存文件
5. 公众号版 → wenyan-mcp 推送
6. 头条版 → 人工粘贴到头条
7. 知乎版 → 人工粘贴到知乎
8. 网站深度版 → md-to-astro.mjs 发布到个人网站
9. 短视频脚本 → 存到 iPhone，剪映制作

## 注意事项

- 每个版本必须原创改写，不能直接复制粘贴同一段文字在不同平台重复发（影响推荐）
- 各平台侧重点不同：
  - 公众号：深度 + 行业洞察
  - 知乎：逻辑 + 数据 + 专业感
  - 头条：钩子 + 短段落 + 易读
  - 网站：完整 + SEO + 收藏价值
  - 短视频：一个爆点讲透
