# AnySearch 中国能源数据刷新模板

## 一句话刷新

在 OpenCode 对话中直接说：

```
用 anysearch 的 batch_search 并行查询：2026年各省电力现货电价、风电光伏装机、碳市场碳价、最新能源政策
```

## 分主题模板

### 💰 刷新现货电价
```
用 anysearch 的 search 查：2026年6月 各省电力现货市场 日前均价 实时均价 广东 山西 山东 蒙西 陕西
用 anysearch 的 search 查：2026年6月 电力现货市场 负电价 省份 甘肃 辽宁 数据
```

### 🌱 刷新装机数据
```
用 anysearch 的 batch_search 并行查询：
2026年 中国 风电 新增装机 各省 内蒙古 新疆 数据
2026年 中国 光伏 新增装机 各省 数据
2026年 中国 新型储能 装机 各省 排名
国家能源局 2026年 上半年 可再生能源 并网 数据
```

### 🏭 刷新碳市场数据
```
用 anysearch 的 batch_search 并行查询：
全国碳排放权交易市场 CEA 碳价 2026年6月 最新
CCER 重启 2026年 签发 项目 价格 走势
碳市场 扩围 水泥 电解铝 钢铁 2026年 进展
```

> **精确数据:** 用 china-ets MCP 获取碳市场精确历史行情
> ```
> 查 china-ets 的 get_market_summary — market=cea
> 查 china-ets 的 query_trading_data — market=cea, start_date=2026-01-01, end_date=2026-06-30
> ```

## 知识库映射

| 数据维度 | 目标知识库文件 | AnySearch 查询 | 替代原始 MCP 函数 |
|---------|--------------|--------------|----------------|
| 现货电价 | 电力市场/赛道文件.md | 电力现货市场 价格 各省 | `china-energy_get_spot_prices` |
| 装机数据 | 赛道总览.md | 风电 光伏 储能 新增装机 | `china-energy_get_national_stats` |
| 碳市场 | 碳市场/赛道文件.md | CEA CCER 碳价 | → 改用 china-ets MCP |
| 政策动态 | 各赛道文件 | 能源 政策 2026 | `china-energy_search_policy` |
| 绿电交易 | 数据中心+能源/赛道文件.md | 绿电 交易 PPA | `china-energy_get_market_data` |
