# 视频 · 分镜制作脚本

**路线：** 全AI视频生成（AI配音 + AI画面 + AI剪辑）
**时长：** 60秒
**工具链：** Kling/即梦（画面）+ 剪映（配音合成+剪辑）
**输出格式：** 竖屏 9:16（视频号/抖音）

---

## 制作流程总览

```
Step 1: 用分镜脚本中的提示词 → 在即梦/Kling逐条生成视频片段（约10分钟）
Step 2: 导出每段视频 → 导入剪映
Step 3: 用剪映"文本朗读"功能 → 选择"沉稳男声"读取配音脚本
Step 4: 对齐视频片段和配音时间轴
Step 5: 添加字幕+背景音乐 → 导出发布
```

## 制作要点

| 要点 | 说明 |
|------|------|
| **配音** | 剪映→文本→朗读→选"沉稳男声"或"解说男声"，语速1.0x |
| **配乐** | 剪映音乐→科技感/纪录片风格→音量-15dB（低于配音） |
| **字幕** | 自动识别后手动修正，字体用黑体，字号适中 |
| **节奏** | 画面切换配合配音节奏，不要太快也不要拖 |
| **封面** | 用Kling生成一张数据中心科技感静态图+标题文字 |

**预计耗时：** AI生成画面15分钟 + 剪映组装15分钟 = 总共30分钟
