# 抖音图文快剪脚本 × 3条

## 什么是"图文快剪"

不用真人出镜，不用数字人。就是：**高清图片/视频素材 + 大字字幕 + 配乐**。剪映免费版就能做，抖音上这种形式流量很好，尤其是知识类内容。

手机操作流程（见底部教程）。

## iPhone剪映 · 图文快剪制作教程（5分钟/条）

### 准备工作
在剪映素材库里搜好以下关键词，各下载3-5个：
- "数据中心" "服务器" "科技感" "芯片"
- "风电" "光伏" "新能源" "电网"
- "城市夜景" "万家灯火"

### 制作步骤

```
1. 打开剪映App → 点「开始创作」
2. 从素材库选一段视频/图片作为底
3. 点「音频」→「音乐」→ 搜推荐BGM
4. 点「文字」→「新建文本」→ 按脚本打字幕
     建议：字体选"现代"或"黑体"，字号拉到最大
           颜色白字+黑色描边（抖音标准）
5. 每条字幕时长拖到对应秒数
6. 每换一个画面：点「画中画」→「新增画中画」→ 选下一个素材
7. 全部拼完后，点「导出」→ 1080p
```
