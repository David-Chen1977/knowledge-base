# 思考流 · 每日记录

**用途：** 记录每天在做的事、看的资料、思考的问题——让知识库"活"起来，不断吸收你的认知输入。

**使用方式：** 每天花 3 分钟，回答下面三个问题即可。

---

## 每日模板

```markdown
# YYYY-MM-DD · 星期X

## 📌 今天做了什么？
<!-- 工作中推进的事项、会议、调研 -->

## 👀 今天看了什么？
<!-- 视频号、文章、研报、新闻 -->

## 💡 今天想了什么？
<!-- 产生的判断、疑问、灵感 -->
```

## 示例

```markdown
# 2026-06-12 · 周五

## 📌 今天做了什么？
- 推进算电协同项目DD，看了两家数据中心公司的材料
- 和团队讨论了电力现货市场改革的影响

## 👀 今天看了什么？
- 视频号：某券商关于算电协同的解读，提到2026年数据中心用电占比将达5%
- 文章：国家能源局《电力市场运行基本规则》征求意见稿

## 💡 今天想了什么？
- 算电协同的核心矛盾不是技术，是利益分配——电网、数据中心、地方政府三方博弈
- 数据中心参与需求响应可能是最快落地的场景，关注虚拟电厂标的
```
