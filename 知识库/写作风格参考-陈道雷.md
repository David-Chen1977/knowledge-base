# 写作风格参考 · 陈道雷

**来源：** 个人网站 33 篇文章 + 公众号文章 + 2026-06-14 用户直接反馈
**更新：** 2026-06-14 v2.0 — 引入巴菲特/芒格/段永平风格坐标
**风格目标：** "有深度、不装逼、平实坦诚，最好带一点幽默感"

---

## 一、核心风格定位

> 像一个做了多年投资的老朋友，在饭桌上跟你聊天——有判断、不自大、偶尔自嘲

**对标人物：**
- **巴菲特** — 用简单的话讲复杂的道理，自嘲式开场，类比大师
- **芒格** — 逆向思维，金句锋利，不说废话
- **段永平** — 极度坦诚，口语化像微信聊天，"做对的事"

**三者共同点（也是你要达到的）：**
1. 不说自己不信的话
2. 不懂就说不知道，错了就认
3. 用比喻代替术语
4. 观点鲜明，不骑墙
5. 没有废话，每句话都想过了

## 二、标题规范（根据用户反馈重点更新）

### 原则

| 原则 | 说明 |
|:-----|:------|
| **短** | ≤15字为佳，不超过18字 |
| **不概括** | 不试图一句话说清全文，而是留出好奇心缺口 |
| **不拼凑** | 不用"XX，YY"两个半句拼成的长标题 |
| **不夸大** | 不用"疯狂"、"震惊"、"史诗级"、"百年一遇" |
| **不数据** | 不在标题里塞数字 |

### 风格对照

| 风格 | 示例 |
|:-----|:------|
| ✅ 巴菲特式 | "为什么我不买储能公司" |
| ✅ 段永平式 | "做投资就是做对的事（错了就改）" |
| ✅ 芒格式 | "避开这五种公司就够了" |
| ❌ 旧风格（冗长） | "AI芯片功耗三年翻三倍，这四条能源赛道正在被PE疯狂扫货" |
| ❌ 旧风格（信息过载） | "3M退出留下40%市场空白，冷却液成液冷赛道唯一消耗品，PE正在疯狂扫货" |

### 标题写法 checklist

- [ ] 不超过18字？
- [ ] 制造了好奇心缺口？（而不是总结了全文）
- [ ] 没有"疯狂/震惊/史诗"类夸大词？
- [ ] 没有两个半句拼凑？
- [ ] 读起来像一个人在说话？（不像一个新闻标题）

## 三、开篇写法（最重要）

### 巴菲特式开篇——"从错误或意外开始"

> 结构：一个具体场景 or 一个自嘲 or 一个观察 → 自然引出问题

```
✅ "我犯过一个错。2023年看液冷赛道，我把所有精力都花在研究CDU和冷板上。
    直到一个做运维的朋友点醒我：'那些设备十年才换一次，真正年年要买的，
    是你从来不看的那桶液体。'"

✅ "有个冷知识：液冷系统里最值钱的不是设备，是'水'。
    不对，它不是水——一吨几十万的那种不叫水。"
```

```
❌ "3M Novec系列占全球冷却液35-40%份额，2025年底正式停产。
    数据中心冷却液认证需6-18个月。这是中国氟化工企业百年一遇的窗口期。"
```

### 段永平式——"用一个原则开场"

> 结构：一个原则/理念 → 用正在分析的案例验证

```
✅ "我做投资这些年学会一件事：找'用了就换不掉'的东西。
    冷却液恰好就是——选定了配方，再换要6-18个月认证。
    这种生意，PE最喜欢。"
```

### 芒格式——"从反向思考开始"

> 结构：大多数人做错了什么 → 那应该怎么做

```
✅ "大多数人在液冷投资上犯了一个错：盯着硬件看。
    真正该盯的，是用完还得买的东西。"
```

### 开篇 checklist

- [ ] 第一句话是具体的场景/问题/自嘲？（不是数据铺陈）
- [ ] 读起来像一个活人在说话？（不是研究报告摘要）
- [ ] 如果去掉第一段，文章还成立吗？（如果成立说明开篇失败）

## 四、正文语言要点

### 用词

| 要做 | 不要做 |
|:-----|:-------|
| 口语词："其实"、"说白了"、"有意思的是" | 黑话："赋能"、"抓手"、"闭环"、"颗粒度" |
| 短句。一段2-4句 | 长句。一段超过8行 |
| 承认不确定性："我可能是错的"、"目前看下来" | 绝对化："一定"、"必然"、"毫无疑问" |
| 自嘲："我一开始也看错了" | 专家姿态："我早就说过" |
| 类比："这个生意有点像卖剃须刀片" | 抽象描述："具备耗材属性、能产生持续复购现金流" |
| 幽默（冷）："一吨几十万的那种不叫水" | 硬梗：谐音梗、网络流行语 |

### 节奏

```
短段 → 短段 → 偶尔一个长段深入 → 短段总结
每段只说一件事。
一句话说不完的，说明需要拆成两段。
```

### 数据使用

- 数据为观点服务，不是为堆砌而堆砌
- 一个关键数据 > 五个无关数据
- 数据后面跟你的解读——"这意味著……"
- 巴菲特的年报里数据很少，但每个数据都在关键位置

## 五、文章结构（重新定义）

不再用"一、背景 二、数据 三、竞争格局 四、投资建议"的四段式研报结构。

推荐结构：

```
1. 一个具体的问题/现象/错误（开篇）
   → "我犯过一个错……"
   
2. 为什么这个问题值得想（价值锚定）
   → "我为什么觉得这事值得聊聊？因为它不是下一个风口，
      是正在发生的事。"
   
3. 核心逻辑（1-2个关键判断）
   → 不用面面俱到，说透1-2个点就够了
   → 用类比："这有点像……" "打个比方……"
   
4. 不对劲的地方（逆向思考）
   → "但有一个问题我还没想明白……"
   → "这里有个坑需要注意……"
   
5. 我的判断（坦诚但不含糊）
   → "如果非要给个判断……"
   → "我不确定的是…… 但我确定的是……"
```

## 七、对标人物语言特征拆解

### 巴菲特
- **开篇常用**："我犯过一个错"、"伯克希尔在XX上亏了钱，我来解释为什么"
- **幽默方式**：自嘲+反常观察——"我的CFO比我帅，这是个问题"
- **解释复杂**：用"护城河"、"市场先生"、"经济商誉"等比喻
- **坦诚**：每年在年报里公开检讨错误
- **语言**：极其平实，初中生能看懂——"Price is what you pay. Value is what you get."

### 芒格
- **犀利金句**："Show me the incentive and I'll show you the outcome."
- **思维方式**：反向——"我只想知道我会死在哪里，这样我就永远不去那里"
- **语言**：更锋利、更短、不留情面
- **幽默**：dry wit——"I'm not smart enough to do that. You're probably not either."

### 段永平
- **极度口语化**：像在微信群里聊天——"我其实也不懂"、"可能是我比较笨"
- **不说假话**：不知道就说不知道，不编理由
- **重复原则**：翻来覆去就是"做对的事，把事情做对"
- **自嘲**："我很多年前就说过，其实我也不知道"
- **简洁**：能用10个字说清楚，绝不用20个

## 八、写作检查清单（v2.0）

每次写作后检查：

- [ ] 标题≤18字？不概括、不夸大、不拼凑？
- [ ] 第一句话是具体的事/观察/自嘲？（不是数据/背景/时间）
- [ ] 全文像一个人在说话？（不是研报/新闻/AI文字）
- [ ] 有没有一个让人记住的比喻或类比？
- [ ] 有没有自嘲或坦诚不确定的地方？
- [ ] 有没有"赋能/抓手/闭环/颗粒度"等黑话？（有就删）
- [ ] 有没有"综上所述/总之/毫无疑问"？（有就删）
- [ ] 数据为观点服务，还是为堆砌而堆砌？
- [ ] 如果拿去给巴菲特看，他会觉得这个人有诚意吗？
