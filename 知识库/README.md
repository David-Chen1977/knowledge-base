# 行业知识库

**用途：** PE/VC 一级股权投资参考知识库，覆盖电力新能源 + AI 两大赛道。
**模式：** 自更新输入层 —— 思考流 → 视频笔记 → 会议纪要 → 赛道深化 → 内容输出

---

## 自更新循环

```
每天 3 分钟输入 → 思考流/ + 视频笔记/
         ↓
 系统自动关联到赛道 → 当前关注.md 驱动优先级
         ↓
 认知积累到阈值 → 触发内容输出（四件套）
         ↓
 产出反哺 → 知识库更新 → 循环
```

## 赛道速查

| 赛道 | 阶段 | 热度 | 一级标的密度 | 政策驱动 |
|------|------|------|-------------|---------|
| 光伏 (新技术方向) | 成熟期局部创新 | ★★★☆☆ | 中（Perovskite/BIPV） | 高（双碳） |
| 储能 | 高速成长期 | ★★★★★ | 高 | 高（新型储能） |
| 风电 (新技术方向) | 成熟期局部创新 | ★★★☆☆ | 中低（浮式/运维） | 高 |
| 氢能 | 早期成长期 | ★★★★☆ | 中高 | 高（氢能规划） |
| 智能电网/VPP | 成长期 | ★★★★☆ | 中 | 高（电改） |
| 碳市场 | 启动期 | ★★★☆☆ | 中 | 高（碳达峰） |
| AI 基础层 | 高速成长期 | ★★★★★ | 高 | 中 |
| AI 模型层 | 整合期 | ★★★★☆ | 中（融资向头部集中） | 中 |
