# 一级市场估值基准（PE/VC 参考框架）

**适用：** 电力新能源 + AI 赛道，中国一级市场（新增 IRA/ITC 政策对中美估值影响分析）
**数据来源：** Prospector Energy MCP（ITC数据）、China Energy MCP（IRR参考）

**注意：** 估值为参考区间，实际情况因团队质量、技术壁垒、收入规模、竞争格局等因素波动很大。

## 一、通用估值逻辑（按发展阶段）

## 四、特殊估值考量（一级股权）

### 当前市场环境（2025-2026）

| 因素 | 影响 |
|------|------|
| 募资难持续 | 估值普降20-40%，人民币基金更谨慎 |
| IPO 收紧 | Pre-IPO 估值倒挂，上市不确定性折价加大 |
| AI 泡沫挤出 | 纯大模型公司面临估值修正，应用层+收入公司获得溢价 |
| 新能源产能过剩 | 制造业环节估值承压，技术壁垒公司更受青睐 |
| 并购退出增加 | 跨境并购、产业资本收购增多，估值逻辑趋向产业整合 |

### 估值调整因子

| 因子 | 溢价(+) / 折价(-) |
|------|------------------|
| 创始团队（顶级连续创业者） | +20~50% |
| 技术壁垒（专利/IP） | +10~30% |
| 国产替代/信创标签 | +15~40% |
| 收入质量（SaaS recurring vs 项目制） | SaaS +20~50% |
| 壁垒型商业模式（网络效应/规模效应） | +30~80% |
| 客户集中度 >50% | -15~30% |
| 政策合规风险 | -20~50% |
| 竞品饱和（超过5家同阶段竞争者） | -20~40% |

## 六、IRA/ITC 税收抵免对估值的影响（2026年新增）

### 美国《通胀削减法案》(IRA) ITC框架

| 抵免类型 | 抵免率 | 条件 |
|---------|--------|------|
| 基础ITC | **30%** | 太阳能、储能、风电等合格技术 |
| 能源社区红利 | +10% | 煤矿关闭区/化石能源依赖区 |
| 低收入社区红利 | +10-20% | 低收入住宅/部落用地项目 |
| 国内含量红利 | +10% | 钢铁/制成品美国本土生产 |
| **最高叠加** | **50%** | 全部条件满足时 |

### ITC对项目估值的影响机制

ITC对一级市场估值的核心影响路径：

1. **资本成本降低** — 30-50%的ITC相当于直接降低项目资本支出，提高IRR
2. **可转让性** — ITC抵免额可出售给第三方（税收股权市场），即"变现"政策红利
3. **估值倍数溢价** — 美国清洁能源项目可比中国同类享受1.5-2x估值倍数，主要来源于ITC
4. **开发商估值逻辑变化** — 开发中项目按"ITC套现后净值"定价，而非传统DCF

### 中美项目IRR/估值对比

| 项目类型 | 中国IRR | 美国IRR（含ITC） | 估值倍数差 | 说明 |
|---------|--------|-----------------|-----------|------|
| 集中式光伏 | 5-8% | 8-15% | 1.5-2x | ITC 30%直接增加税后收益 |
| 储能（独立） | 4-7% | 8-12% | 1.5-2.5x | 储能ITC资格是IRA重大利好 |
| 陆上风电 | 6-9% | 7-12% | 1.2-1.8x | PTC替代ITC，production-based |
| 分布式光伏 | 6-10% | 12-20% | 2x+ | 加ITC+净计量+SREC三重收益 |

### 对PE/VC投资的启示

| 启示 | 说明 |
|------|------|
| 美国项目有系统性政策溢价 | 在估值基准中应+30-50%政策因子 |
| ITC可转让性创造流动性 | 项目公司可通过税收股权合作提前回笼ITC价值，改善现金流 |
| 需关注IRA政治风险 | 2026年中期选举后IRA条款可能调整，影响远期项目估值 |
| 中国项目估值需考虑碳收益 | 全国碳市场扩围+CCER重启将为中国项目增加碳收益估值维度 |
| 交叉投资机会 | 中国技术+美国项目：如中国光伏组件+美国电站（需注意本土化要求） |
