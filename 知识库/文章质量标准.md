# 文章质量标准 v2.0

> 三级质量体系：合格线 → 优秀线 → 卓越线。
> 合格线是底线，卓越线是追求。
> 每一篇都至少过合格线；每三篇至少有一篇摸到优秀线。
> 卓越线是用来逼自己的，不是每篇都要达到。

---

## 第一级：合格线（必过 · 不发则已，发则必过）

> 不过这一级，不发。

### 标题

- [ ] **≤15字？** 超了重写
- [ ] **不是句子拼凑？** "XX，YY"式拼凑标题 → 重写
- [ ] **无夸大词？** "疯狂、震惊、史诗级、百年不遇、抄底" → 删
- [ ] **不是新闻标题体？** 不像"XX事件引发XX关注" → 标题应该像一个人在说话
- [ ] **制造好奇心而不是概括全文？** 如果标题把内容说完了，读者没有点进来的理由

### 开篇（前3段）

- [ ] **第一句是具体的事/观察/反常/问题？** 不能以"近年来""随着""在当前"开头
- [ ] **没有编造的故事/对话/经历？** 只要是编的，文字功底多好都不行
- [ ] **不是数据堆砌？** 开头不堆3个以上数据

### 正文

- [ ] **有清晰的可记忆的框架？** 读者读完后能复述出"两条逻辑""三种模式""四个方向"
- [ ] **有明确的个人判断？** 不是"行业分析报告"，必须有"我认为""我的判断是""我比较确定的是"
- [ ] **没有黑话？** "赋能、抓手、闭环、颗粒度、底层逻辑、范式转移" → 出现了就没过
- [ ] **没有废话填充？** "综上所述、总之、值得关注的是、不可忽视的是" → 删
- [ ] **承认不确定性？** 至少有一处说"这个我还没想清楚"或"我可能是错的"
- [ ] **数据不孤立？** 每个数字后面跟一句话说明"这意味着什么"

### 结尾

- [ ] **有余味？** 不是总结上文，而是让读者带走一个思考

### 格式

- [ ] **无Markdown残留？** 微信草稿箱预览检查一遍
- [ ] **无AI味排比句？** "不是A也不是B而是C"句式最多出现一次
- [ ] **公众号：每段不超过4行（手机上）？** 否则阅读体验差
- [ ] **网站：数据来源已标注？**

## 第二级：优秀线（追求 · 三篇至少一篇达到）

> 这一级是对标行业内头部内容的标准。

### 独特性

- [ ] **有至少一个"在这篇文章之前我没见过"的洞察或框架？** 不是从研报里抄来的
- [ ] **有一个让人印象深刻的比喻/类比？** 读者能记住它，甚至转述给别人（如"淘金热里卖铲子的人"）
- [ ] **观点有锋芒？** 不是"一方面……另一方面……"的骑墙文。读者看完知道"作者是站在哪一边的"

### 证据链

- [ ] **每个关键判断有至少两个独立来源支撑？** 不依赖单一数据源
- [ ] **有反方观点并回应了它？** "可能会有人反驳说……我的看法是……"
- [ ] **不是二手信息堆砌？** 至少有1-2个数据是你从原始来源（SEC/能源局/公司公告）里自己挖的

### 表达

- [ ] **非专业读者也能看明白核心逻辑？** 如果只懂投资不懂技术也能看懂，过
- [ ] **至少有一句"可以引用的金句"？** 一句话说明核心观点，像"Price is what you pay, value is what you get"那样的密度
- [ ] **读起来有节奏感？** 长短句交替。短句收尾。不全是陈述句

### 传播力

- [ ] **朋友转发的理由是什么？** 想清楚：读者为什么要把这篇文章发给别人？信息增量？观点认同？社交货币？
- [ ] **标题能让人在朋友圈点开？** 不是标题党，但要有足够的信息密度让人好奇

## 第三级：卓越线（追求 · 十篇至少一篇）

> 这一级对标的是"行业里提起这个话题，首先想到的那篇文章"。

### 原创性

- [ ] **提出或者推广了一个可命名的概念/框架？** 像"非均衡理论"、"市场的逻辑"、"算电协同"——别人之后讨论这个领域时会引用你的说法
- [ ] **包含一手数据/独家信息？** 不是只有公开信息，有你自己挖掘或整理的独家数据
- [ ] **做出了可验证的推理？** 不判断"市场可能会增长"，而是"A条件达成 → B结果会以C速度发生"——事后可以验证

### 思想深度

- [ ] **超越了"是什么"和"怎么样"，触及了"为什么"？** 不只是描述现象，而是解释了底层的驱动力
- [ ] **有明显的"厉以宁/张维迎"思维痕迹？** 非均衡视角（看到别人没看到的不均衡，机会就在那里）或清晰二分法框架（"两种逻辑""两条路"）
- [ ] **读者读完会觉得自己的认知被提升了一个层次？** 不是"又知道了一个消息"，而是"看问题的角度变了"

### 影响力

- [ ] **行业内有影响力的人会愿意转发/引用？** 不是为了大众阅读量写的，而是为了影响有影响力的人
- [ ] **一个月后还有人提起这篇文章？** 不是"看完了就忘了"，而是成为该话题的一个参考点
- [ ] **你自己三个月后回看，不会觉得"这写得不行"？** 能经得起时间检验

### 表达

- [ ] **每个字都值它在文章中的位置？** 删掉任何一段，文章都会变差——没有一句废话
- [ ] **有巴菲特/芒格式的智慧密度？** 不是堆砌聪明话，而是几条简单但深刻的原则贯穿全文
- [ ] **读起来是一种享受？** 即使读者不同意你的观点，也会享受阅读的过程本身

---

## 四、分级使用说明

```
合格线：每篇必过 → 不过不发
优秀线：每三篇至少一篇达到 → 否则说明没有在进步
卓越线：每十篇至少一篇追求 → 这是出代表作的标准
```

**操作流程：**
1. 写完初稿 → 合格线逐条检查
2. 不通过 → 修改 → 重检 → 直到全过
3. 通过合格线 → 问：这篇能不能摸到优秀线？
4. 如果能 → 按优秀线标准打磨，至少满足70%条目
5. 如果感觉特别好 → 再看看卓越线

**迭代规则：**
- 每次发现新的质量问题，加入对应级别
- 每次达到卓越线，在进步日志中记录
- 每个季度回顾一次标准，看是否需要提高
