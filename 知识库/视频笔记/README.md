# 视频号笔记 · 知识捕获

**用途：** 看视频号时快速记录关键信息，让"看过"变成"学过"。

**使用方式：** 看视频时随手记，30 秒一条。

---

## 笔记模板

每次看完一个视频，新建文件 `YYYY-MM-DD-主题.md`：

```markdown
# YYYY-MM-DD · 视频标题

## 来源
- **平台：** 视频号
- **作者/账号：** 
- **链接：** （可选）

## 关键信息
<!-- 3-5个核心点 -->

## 我的判断
<!-- 同意/不同意？为什么？和我的认知有什么关系？ -->

## 关联赛道
<!-- 算电协同 / 储能 / AI应用 / 光伏 / ... -->
```

---

## 示例

```markdown
# 2026-06-12 · 算电协同：AI时代的电力瓶颈

## 来源
- **平台：** 视频号
- **作者/账号：** 某券商首席经济学家

## 关键信息
1. 2026年中国数据中心用电量预计达 3000 亿 kWh，占全社会用电量 3.5%
2. 算电协同的核心是"源网荷储"一体化
3. 政策窗口期：电改 2.0 推动电力市场化交易

## 我的判断
- 3000 亿 kWh 这个数据比之前看的偏低（之前有说 5%），需要交叉验证
- 源网荷储一体化的商业模式还不清晰，谁出钱是个问题

## 关联赛道
算电协同 > 电力市场
```

## 自动化管道（2026-06-15 新增）

### iPhone 快捷指令"保存视频笔记"

```
看视频 → 分享 → "保存视频笔记"
  → 自动捕捉链接+标题
  → 存入 iCloud Drive/workbuddy-inbox/
  → Mac inbox-watcher 自动接收
  → 下次对话时 AI 自动填充结构化笔记
```

### 创建快捷指令（iPhone，3分钟）

1. 打开**快捷指令** App → **+** → 命名 "保存视频笔记"
2. 添加动作：
   ```
   ① "获取剪贴板"（视频分享时自动复制链接）
   ② "获取URL内容"（抓取页面信息）
   ③ "存储文件" → iCloud Drive/workbuddy-inbox/视频_{当前日期}.md
   ④ "显示通知" → "✅ 视频笔记已保存"
   ```
3. 开启 **"在分享表中显示"**
